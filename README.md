Personal Intro Program

A beginner Python program that asks a user several questions and prints a formatted user profile.

Description

This program collects information from the user such as their name, location, language, workplace, college, hobbies, and preferred programming language. After gathering the input, the program displays the responses in a formatted “User Profile” section.

This project was built as part of learning Python fundamentals.

## Concepts Used

- Python variables
- `input()` for user interaction
- String concatenation
- Formatted console output
- Basic program structure

## How to Run the Program

1. Open a terminal.
2. Navigate to the project folder.
3. Run the program:

python intro.py

or

python3 intro.py
